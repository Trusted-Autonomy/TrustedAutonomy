# Start Your First Goal

Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) and run **TA: Start New Goal**.

## Examples

- `Add input validation to the login form`
- `Write unit tests for the user authentication module`
- `Fix the N+1 query problem in the posts list endpoint`
- `Refactor the payment service to use dependency injection`

## What happens next

1. The agent starts working in an isolated staging workspace
2. The **TA Goals** sidebar shows the goal as **running**
3. A notification appears when the draft is ready to review
4. The **TA Drafts** sidebar shows the pending draft

The agent works autonomously but **never modifies your project** until you explicitly approve its draft.
