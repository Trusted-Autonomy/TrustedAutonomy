# Review and Approve Drafts

When the agent finishes, a notification appears and the **TA Drafts** panel shows the pending draft.

## Reviewing a draft

1. Click the draft in the **TA Drafts** panel
2. Right-click → **View Diff** to open each changed file in VS Code's native diff editor
3. The left side shows the original file; the right side shows the agent's proposed changes

## Approving

Right-click a draft → **Approve Draft**. A confirmation dialog appears before any changes are applied to your project.

## Denying

Right-click a draft → **Deny Draft**. You'll be prompted to enter a reason. The agent can see this reason when you start a follow-up goal to address your feedback.

## Opening the full review UI

Click **TA: Open Shell** (status bar or Command Palette) to open the TA web shell in your browser. It provides a richer review interface with side-by-side diffs, comments, and the full change summary.
